# Dell Inventory Management System

## Project Overview
The Dell Inventory Management System is a simple web-based application developed using HTML, CSS, and JavaScript. It helps users manage Dell product inventory by adding products, checking stock availability, viewing inventory, and removing products.

## Features
- Display available Dell products
- Add new products to inventory
- Check product availability
- Remove products from inventory
- Dynamic inventory updates using JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure

Dell-Inventory-System/
│
├── index.html
├── style.css
├── script.js
└── README.md

## Functionalities

### Add Product
Users can add a new product to the inventory list.

### Check Availability
Users can check whether a product is available in stock.

### Remove Product
Users can remove a product from the inventory.

### Display Inventory
All available products are displayed dynamically on the webpage.

## Sample Products
- Dell Inspiron
- Dell XPS
- Dell Latitude
- Dell Alienware

## How to Run

1. Download the project files.
2. Open `index.html` in any web browser.
3. Use the interface to manage inventory.

## Output
The application displays:
- Product inventory list
- Stock availability status
- Updated inventory after adding or removing products
